# 莘睿未来 HKDSE 物理学习平台

> Shine Ray Future - HKDSE Physics Learning Platform

一个现代化的 HKDSE 物理学习平台，帮助香港学生更好地掌握物理知识。

## ✨ 功能特点

### 📚 八大物理主题
- 🌡️ Temperature and Gas（温度与气体）
- 🚀 Force and Motion（力学与运动）
- 🌊 Wave Motion（波动）
- ⚡ Electricity and Magnetism（电与磁）
- ☢️ Radioactivity and Nuclear Energy（放射性与核能）
- 🌌 Astronomy and Space Science（天文与太空科学）
- ⚛️ Atomic World（原子世界）
- 💡 Energy and Use of Energy（能量及能量的使用）

### 🎯 每个主题包含
- **理论学习模块** - 详细的概念讲解和关键要点
- **互动模拟演示** - 可视化物理现象，支持参数调整
- **公式计算器** - 输入已知量计算未知量
- **练习题** - 多种题型，即时反馈

### 👤 用户系统
- 学习进度追踪
- 个性化学习推荐
- 成就徽章系统
- 经验等级体系

### 📝 评估系统
- 章节测验
- 模拟考试
- 错题本复习

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **React 18** - 用户界面库
- **TypeScript** - 类型安全
- **Tailwind CSS** - 原子化 CSS 框架
- **Vite** - 现代化构建工具
- **Framer Motion** - 动画库
- **Zustand** - 状态管理
- **React Router** - 路由管理
- **Recharts** - 数据可视化
- **Lucide React** - 图标库

## 📁 项目结构

```
src/
├── components/        # 可复用组件
│   ├── Layout/       # 布局组件
│   └── UI/           # UI 基础组件
├── constants/        # 常量和数据
├── pages/            # 页面组件
├── store/            # 状态管理
├── types/            # TypeScript 类型定义
├── App.tsx           # 应用入口
├── main.tsx          # 渲染入口

```

## 📖 使用说明

1. **首页（Dashboard）**
   - 查看学习统计和进度
   - 快速访问各个主题
   - 查看今日任务和成就

2. **主题学习页面**
   - 选择章节进行学习
   - 切换理论/模拟/计算/练习标签
   - 完成练习题获得经验

3. **测验系统**
   - 章节测验：巩固单章知识
   - 模拟考试：检验整体水平

4. **成就系统**
   - 完成学习任务解锁成就
   - 提升等级获得称号

5. **错题本**
   - 自动收录做错的题目
   - 支持按主题筛选
   - 标记已掌握的题目
