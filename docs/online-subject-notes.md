## 3) Radiation and Nuclear

### Chapters

- radiation-radioactivity (84)
- atomic-model (46)
- nuclear-energy (45)

### Core formulas

- `N = N0 e^(-lambda t)`
- `A = lambda N`
- `A = A0 e^(-lambda t)`
- `lambda = ln2 / T1/2`
- `N = N0 (1/2)^(t/T1/2)`
- Corrected count rate: `R = Rtotal - Rbg`
- `E = Delta m c^2`
- `1 u = 931 MeV`

### Common question types

- Half-life from count-rate data (with background correction)
- Identify alpha/beta/gamma from deflection/penetration
- Nuclear equation balancing
- Fission/fusion and chain reaction conditions

### Frequent mistakes

- Forgetting background subtraction
- Treating half-life as linear decay
- Wrong atomic number/mass number updates
