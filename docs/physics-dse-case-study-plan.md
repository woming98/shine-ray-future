# DSE 物理知识专题库生产规范

## 目标

把 DSE Physics 2021-2025 真题做成可长期扩展的知识专题库。每个独立 HTML 页面以一个知识体系为中心：先把概念、公式、图像语言和常见考法讲透，再引入对应真题作为验证案例。页面不放原题截图，不复制完整题面，只使用原创教学摘要、重画图和独立解析。目标读者是有一定物理底子、数学能力强但长期没有复习的人：看完后能快速恢复模型感，知道这个知识点如何被 DSE 命题，并能迁移到同类题。

## 当前样板

已先完成 2024 Paper 1 前 5 个知识专题样板：

- 加热曲线：heating curve 与 specific heat capacity
- 冰水混合：water-ice mixture 与 latent heat energy audit
- 极低压气体：ideal gas 与 molecule count order of magnitude
- 运动图像：velocity-time graph 的 shaded area
- 粗糙斜面：rough incline 上匀速下滑时的 contact resultant

入口：`/physics/dse-2024/index.html`

## 单个知识专题页面标准

每个专题至少包含以下模块：

- 先建立知识体系：核心思想、学习地图、知识边界
- 核心知识讲透：概念、公式、图像语言和常见误解
- DSE 常见考法：这个知识点通常如何被命题
- 必要前置知识：长时间没复习的人需要先恢复的旧知识
- 2024 真题入口：重构题意、研究问题、快速结论和重画图
- 真题推理路径：从知识点到答案的考场路径
- 完整解法：不跳关键步骤
- 错选项为什么错：逐项解释错选项的诱因
- 拿分点地图：拿分点/判断点
- 高分陷阱：高分学生也会踩的陷阱
- 高分反射：以后遇到同类题的第一反应
- 变式练习：一道变式检查是否真正掌握

## 生产流程

1. 建知识专题清单：先按热学、力学、气体、图像、波、电等知识体系归类，再挂接年份、paper、题号、答案、官方得分率。
2. 解析题目：只记录必要物理条件和考点，不复制完整题面。
3. 写知识体系：先讲概念、公式、图像语言、常见误解和 DSE 命题方式。
4. 画图重构：凡是依赖图像理解的题，重新画一张教学图，突出变量、方向、关键区域。
5. 写真题验证：把 2024 真题放在知识专题后半段，用来验证这个知识如何被考。
6. 录入结构化内容：新增到 `content/physics-dse/<year>/p1/cases.ts` 或后续对应年份文件。
7. 生成静态页：运行 `npm run generate:physics-cases`。
8. 验收：跑 `npm run build`，并检查每个独立 HTML URL 返回 200。

## 验收清单

- 不出现原题截图、原卷整页图、考试机构水印图。
- 不照搬完整题面，只保留原创教学化摘要。
- 答案与 marking scheme 一致。
- 图像是重画图，并且比原题更适合学习。
- 页面开头必须先讲知识体系，不能一上来做题。
- 解释能让“有基础但久未复习”的学生恢复模型。
- 每题能独立打开，不依赖 React 路由。
- 单题页面有变式练习，能检查迁移能力。
