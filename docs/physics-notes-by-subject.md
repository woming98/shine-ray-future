### Core formulas frequently used
- `V = IR`
- `P = VI = I^2R = V^2/R`

- Series/parallel resistance rules
- Transformer: `Vp/Vs = Np/Ns`
- Ideal transformer power: `VpIp = VsIs`
- Efficiency: `eta = P_out / P_in`

### High-frequency question patterns
- Multi-step transformer + efficiency + current chain

- Induced current direction by Lenz’s law
- AC rms/peak conversion and heating equivalence

### Common traps
- Confusing turns ratio direction
- Using DC assumptions on transformer questions
- Ignoring cable resistance in line-loss questions
- Wrong current direction when combining field + motion

## 3) Radiation & Nuclear (175)

### Chapters
- `radiation-radioactivity` 84
- `atomic-model` 46
- `nuclear-energy` 45

### Core formulas frequently used
- Decay law: `N = N0 e^(-lambda t)`
- Activity: `A = lambda N`, and `A = A0 e^(-lambda t)`
- Half-life relation: `lambda = ln2 / T_half`
- Half-life shortcut: `N = N0 (1/2)^(t/T_half)`
- Background-corrected count: `R_corrected = R_total - R_bg`
- Nuclear equation balancing (mass no. and atomic no.)
- Mass-energy: `E = Delta m c^2`
- Unit conversion: `1 u = 931 MeV`

### High-frequency question patterns
- Count-rate/half-life extraction with background subtraction
- Identify alpha/beta/gamma from deflection + penetration data
- Multi-step nuclear decay bookkeeping
- Fusion/fission classification and chain reaction conditions
- Mass defect and energy release estimate

### Common traps
- Using total count instead of corrected count
- Treating half-life as linear decay
- Wrong alpha/beta effect on mass and atomic numbers
- Sign mistakes in electric/magnetic deflection logic

## 5) Atomic World (32)

### Chapter
- `atomic-world` 32

### Core formulas frequently used
- `E = hf = hc/lambda`
- Photoelectric equation: `hf = phi + K_max`
- Stopping potential: `K_max = eVs`
- Threshold frequency: `phi = hf0`
- Photon rate-power relation: `P = (N/t)hf`

### High-frequency question patterns
- Work function and threshold frequency extraction
- Graph interpretation (`K_max-f`, `Vs-f`, intensity effects)
- Distinguishing effects of frequency vs intensity

### Common traps
- Assuming intensity changes `K_max`
- Assuming frequency doubling doubles electron speed directly
- Confusing current change with energy-per-electron change

## 6) Energy & Use of Energy (51)

### Chapter
- `energy-and-use-of-energy` 51

### Core formulas frequently used
- Illuminance (point source): `E = Phi/(4pi r^2)`
- Lambert cosine correction: `E = [Phi/(4pi r^2)] cos(theta)`
- Luminous flux from efficacy: `Phi = (lm/W) x P`
- Efficiency: `eta = useful/input`
- Thermal energy: `Q = mc Delta T`
- Cost: `cost = (E in kWh) x tariff`
- Heat pump COP: `COP = Q_H/W`
- Refrigerator COP: `COP = Q_C/W`

### High-frequency question patterns
- Lamp comparison by efficacy and lifetime cost
- Real-world appliance efficiency and yearly energy labels
- Geometry-heavy illumination questions (distance + angle)
- Cooking/heating system efficiency comparison

### Common traps
- Forgetting geometric cosine factor
- Mixing `W`, `Wh`, `kWh`, and `J`
- Treating COP as efficiency percentage
