## 2. 力学总公式表（先背）

### 2.1 运动学（匀加速）

- `v = u + at`
- `s = ut + (1/2)at^2`
- `v^2 = u^2 + 2as`
- `a = Delta v / Delta t`

### 2.2 牛顿定律与受力

- `F_net = ma`
- `W = mg`
- 平衡条件（平动）：`Sigma F = 0`

### 2.3 力矩与平衡

- `M = Fd`（d 为垂直力臂）
- 转动平衡：`Sigma M_clockwise = Sigma M_anticlockwise`

### 2.4 功、能、功率

- `W = Fs cos(theta)`
- `P = W/t = Fv`
- `Ek = (1/2)mv^2`
- `Ep = mgh`
- 机械能守恒（无非保守力做功）

### 2.5 动量与冲量

- `p = mv`
- `J = F Delta t = Delta p`
- 动量守恒：`Sigma p_before = Sigma p_after`

### 2.6 抛体与圆周

- 抛体分解：`ux = u cos(theta)`, `uy = u sin(theta)`
- 竖直方向：`y = uy t - (1/2)gt^2`
- 水平方向：`x = ux t`
- 向心加速度：`a_c = v^2/r`
- 向心力：`F_c = mv^2/r`

### 2.7 万有引力

- `F = Gm1m2/r^2`
- `g = GM/r^2`

## 3. 分章高频考点

## 3.1 Position and Movement（运动学）

**高频题型**

- `s-t`、`v-t`、`a-t` 图像互判
- “第 n 秒位移”、平均速度/平均速率区分
- 匀加速多阶段运动（先加速后减速）

**易错点**

- 把平均速度当平均速率
- 图像面积/斜率意义混淆
- 正负方向约定不一致

---

## 3.2 Newton's Laws（牛顿定律）

**高频题型**

- 水平面摩擦+拉力系统
- 斜面受力分解
- 两物体连接（绳、滑轮）系统加速度
- 第三定律作用力/反作用力辨认

**易错点**

- `F=ma` 用了“单个力”而不是“合力”
- 把作用力与平衡力混为一类
- 漏画法向力或摩擦方向判断错

---

## 3.3 Moment of Force（力矩）

**高频题型**

- 杠杆平衡与支点反力
- 不同位置加力求最小力
- 梯子/杆件静力平衡

**易错点**

- 力臂不是“到支点距离”，而是“到作用线垂距”
- 顺时针/逆时针符号混乱

---

## 3.4 Work, Energy and Power（功、能、功率）

**高频题型**

- 功能关系与机械效率
- 功率比较（同路程、同时间、同速度）
- 重力势能与动能互转

**易错点**

- `W = Fs` 忽略夹角，漏掉 `cos(theta)`
- 机械能守恒条件误用（有摩擦仍硬套）
- 功率单位换算错误（W、kW）

---

## 3.5 Momentum（动量）

**高频题型**

- 碰撞（弹性/非弹性）与爆炸反冲
- 冲量-时间图像题
- 安全装置（延长碰撞时间减小平均力）

**易错点**

- 忽略方向导致动量代数和错误
- 把“动量守恒”用于有明显外力系统
- 冲量算成 `F/t`（应为 `F Delta t`）

---

## 3.6 Projectile Motion（抛体运动）

**高频题型**

- 射程、飞行时间、最高点
- 同时性问题（落地时间与水平位移）
- 与自由落体联动比较

**易错点**

- 不分解就硬代公式
- 水平分量误设有加速度
- 角度分解 sin/cos 写反

---

## 3.7 Circular Motion（圆周运动）

**高频题型**

- 水平圆周（绳球、转盘）
- 竖直圆周（最低点/最高点受力）
- 周期、角速度与线速度关系

**易错点**

- 把向心力当“新力种”（它是合力方向）
- 最高点受力条件判断错误
- 匀速圆周误判“加速度为零”

---

## 3.8 Gravitation（万有引力）

**高频题型**

- 行星轨道半径变化对 `F`、`g` 影响
- 比值题（距离变 n 倍，力变 1/n^2）
- 重力场强与高度关系

**易错点**

- 反平方关系当一次反比
- 质量与重量混用

---

## 4. 力学通用解题流程（考试版）

1. **先定方向**：先规定正方向，再列式。  
2. **画图**：运动图 + 受力图（FBD）必须画。  
3. **选模型**：匀加速 / 平衡 / 动量守恒 / 能量守恒。  
4. **列核心方程**：最多 2~3 条主方程联立。  
5. **检查单位与符号**：m/s、N、J；方向符号是否一致。  
6. **数量级复核**：结果是否明显离谱。  

---

## 5. 临考速记（高频陷阱）

- `v-t` 图下面积是位移，不是路程（有负值时要小心）。  
- 平均速度看位移，平均速率看路程。  
- `F=ma` 永远用合力。  
- 力矩看垂直力臂，不看斜边长度。  
- 功能题先判有没有非保守力（摩擦、阻力）。  
- 动量题先判系统是否近似“外力冲量可忽略”。  
- 抛体水平与竖直分开做。  
- 圆周题“向心力”是指向圆心的合力。  
- 引力题几乎都离不开反平方。  
- 每一步都带单位，能少错至少一半。
