## 2) 光学总公式（先背）

## 2.1 基础波动

- `v = f lambda`

## 2.2 反射与折射

- 反射定律：`i = r`
- 斯涅尔定律：`n1 sin(theta1) = n2 sin(theta2)`
- 折射率：`n = c/v`
- 相对折射率关系：`n = sin(i)/sin(r)`（空气到介质）
- 临界角：`sin(c) = 1/n`（由密到疏）
- 视深关系：`n = real depth / apparent depth`

## 2.3 透镜

- 薄透镜公式：`1/f = 1/u + 1/v`
- 放大率：`m = v/u = image height / object height`

## 2.4 干涉与衍射

- 双缝条纹间距：`Delta y = lambda D / a`
- 光栅方程：`d sin(theta) = n lambda`
- 干涉条件：  
  构成加强：`Delta x = n lambda`  
  构成减弱：`Delta x = (n + 1/2) lambda`
- 相位差：`phi = 2pi Delta x / lambda`

## 3) 分章高频考点

## 3.1 光的传播（Propagation）

**高频题型**

- 光沿直线传播的几何作图
- 光速、频率、波长三者转换
- 不同介质中速度变化与频率不变

**易错点**

- 误以为进介质后频率改变（实际频率由光源决定）
- 单位不统一（nm、um、m）

---

## 3.2 反射与折射（Reflection & Refraction）

**高频题型**

- 入射角/反射角关系
- 折射角计算与折向判断（向法线/离法线）
- 临界角与全反射判定
- 视深、折射率反推

**易错点**

- 角度基准用错：应对“法线”量角，不是对界面
- 全反射条件写不全：  
  1) 由密到疏；2) 入射角大于临界角
- 视深题把分子分母写反

---

## 3.3 透镜（Lens）

**高频题型**

- 已知物距像距求焦距
- 放大率、像高、像距联立
- 实像/虚像、正立/倒立、放大/缩小判定
- 作图题（主光线）

**易错点**

- 凸透镜与凹透镜符号约定混乱
- 像距正负号使用不一致
- 只代公式不做“成像性质”合理性检查

---

## 3.4 光的波动性（Wave Nature of Light）

**高频题型**

- 电磁波谱排序与波长频率关系
- 双缝干涉条纹变化趋势（改 `lambda`、`D`、`a`）
- 衍射显著性比较（孔径与波长同量级）
- 光栅级次与角度判断

**易错点**

- 把亮纹/暗纹条件对调
- 认为“波长变长，条纹更密”（实际更疏）
- 光栅中级次 `n` 与最大可见级次判断错误

---

## 3.5 波动现象中的光学题（Wave Phenomena, optics-related）

**常见混合题**

- 同相/反相双源的干涉判定
- 先求程差再判明暗
- 干涉与衍射概念区分

**易错点**

- 题干给“反相源”仍套同相条件
- 用“单条路径长度”代替“程差”

## 4) 光学解题模板（考试可直接套）

## 模板 A：折射/全反射

1. 先画法线，标 `i`、`r`  
2. 判断介质疏密（看 n 或已知材料）  
3. 用 `n1 sin(theta1)=n2 sin(theta2)`  
4. 若问全反射，再检验两条件

## 模板 B：透镜计算

1. 写 `1/f = 1/u + 1/v`  
2. 写 `m = v/u = hi/ho`  
3. 联立求未知  
4. 最后检查像的性质是否合理

## 模板 C：干涉条纹

1. 判同相/反相  
2. 求程差 `Delta x`  
3. 套亮暗条件  
4. 若问条纹间距，再用 `Delta y = lambda D / a`

## 模板 D：光栅

1. 用 `d sin(theta) = n lambda`  
2. 注意 `sin(theta) <= 1` 判断可见级次  
3. 多波长比较：`lambda` 越大，`theta` 越大

## 5) 临考速记（高频陷阱）

- 所有角度都对法线量。  
- 入射到介质：频率不变，速度和波长改变。  
- 全反射必须“由密到疏 + i > c”。  
- `n = c/v`，介质越“光密”，`n` 越大、`v` 越小。  
- 透镜题一定做“实虚、正倒、大小”复核。  
- 干涉题先看同相/反相再判亮暗。  
- 条纹间距与 `lambda`、`D` 成正比，与 `a` 成反比。  
- 光栅题先检查 `n` 是否超最大级次。
