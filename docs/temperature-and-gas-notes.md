## 二、总览公式表

- 热学基础：`Q = mc Delta T`
- 潜热：`Q = mL`
- 电热：`E = Pt`
- 理想加热（忽略热损失）：`Pt = mc Delta T`
- 热容量：`C = mc`，且 `Q = C Delta T`
- 混合平衡（无损）：`m1 c1 (T1 - Tf) = m2 c2 (Tf - T2)`
- 波义耳定律（等温）：`p1 V1 = p2 V2`
- 查理定律（等压）：`V1/T1 = V2/T2`（T 用 K）
- 压强定律（等容）：`p1/T1 = p2/T2`（T 用 K）
- 综合法则：`p1 V1 / T1 = p2 V2 / T2`
- 理想气体方程：`pV = nRT`
- 分子均方根速率：`c_rms = sqrt(3RT/M)`
- 分子动理论方程：`pV = (1/3) N m c_rms^2`

## 三、分章笔记

## 1) 温度、热、内能（46）

### 核心点

- 区分“热”和“内能”：  
  `热`是能量传递过程，`内能`是系统储存的微观能量。
- 温度变化计算核心：`Q = mc Delta T`
- 温差换算：`Delta T` 的数值在 `deg C` 和 `K` 中相同。

### 高频题型

- 比热容比较（同 Q 下比较 `m Delta T`）
- 电热器加热液体（`Pt = mc Delta T`）
- 温度计线性刻度/校准题
- 混合后平衡温度

### 易错点

- 把 `Delta T` 误写成 `T + 273`
- 混合题漏掉“容器/环境”吸热
- 把“功率大”误判为“温升一定快”（要看质量和比热）

---

## 2) 物态变化（85）

### 核心点

- 相变吸放热：`Q = mL`
- 相变时温度通常保持不变（熔化、沸腾阶段）
- 加热曲线分段处理：升温段用 `mc Delta T`，平台段用 `mL`

### 高频题型

- 冰到水到蒸汽的分段总能量
- 比较熔化潜热与汽化潜热
- 冷却曲线/加热曲线判读

### 易错点

- 平台段仍用 `mc Delta T`
- 忘记单位统一（g 转 kg）
- 相变方向和符号搞反（吸热/放热）

---

## 3) 气体定律（50）

### 核心点

- 先判过程：等温 / 等压 / 等容，再选公式
- 绝对温度必须用 `K`
- 可变过程优先用：`p1 V1 / T1 = p2 V2 / T2`

### 高频题型

- 活塞压缩、气球受热膨胀
- 图像题（`p-V`, `V-T`, `p-T`）
- 比例题与单位换算题

### 易错点

- 直接用摄氏温度做比例
- 过程条件看错（例如误把等容当等压）
- 体积单位混用（L 与 m^3）

---

## 4) 分子动理论（50）

### 核心点

- 理想气体假设：分子体积可忽略、分子间作用力可忽略、碰撞弹性
- 温度越高，分子平均动能越大
- `c_rms` 与 `sqrt(T)` 成正比，与 `sqrt(1/M)` 成正比

### 高频题型

- 同温不同气体的速率比较
- 压力来源解释（分子碰撞器壁）
- 平均动能变化百分比题

### 易错点

- 认为“每个分子”动能都相同
- 用摄氏温度直接做 `sqrt(T)` 比例
- 把“分子数密度相同”误判成“质量密度相同”

---

## 5) 热传递（39）

### 核心点

- 传导（conduction）：靠微观碰撞传能
- 对流（convection）：流体整体流动传能
- 辐射（radiation）：电磁波传能，可在真空中进行

### 高频题型

- 保温瓶、衣物、建筑隔热设计
- 海陆风、烟囱、暖空气上升机制
- 黑色/银色表面对吸收与发射的影响

### 易错点

- 误以为真空能“传导”热
- 混淆“好吸收体”和“好发射体”
- 只考虑一种传热机制，忽略复合过程

## 四、常用解题模板

## 模板 A：加热/降温计算

1. 写已知：`m, c, Delta T` 或 `P, t`
2. 计算输入能量：`E_in = Pt`（若有电热器）
3. 计算有用能量：`Q = mc Delta T`
4. 若有损耗：`E_loss = E_in - Q`

## 模板 B：相变分段

1. 画流程：升温 -> 相变 -> 再升温  
2. 每段分别算：`mc Delta T` 或 `mL`
3. 最后求和

## 模板 C：气体状态变化

1. 判断过程条件（等温/等压/等容/一般）
2. 温度先转 K
3. 代入对应气体方程求未知

## 模板 D：混合平衡温度

1. 设平衡温度 `Tf`
2. 写“放热 = 吸热”（必要时加环境损失项）
3. 解方程并检查 `Tf` 是否在两初温之间

## 五、临考速记（10 条）

- `Delta T` 不用加 273；`T` 做比例一定用 K。  
- 相变平台温度不变，但能量在变化。  
- `Q = mc Delta T` 与 `Q = mL` 不可混用。  
- 电热题先看是否“忽略热损失”。  
- 混合题先判断谁放热谁吸热。  
- 气体题先判过程再选公式。  
- `c_rms` 只和 `T`、`M` 有关。  
- 真空中只有辐射，没有传导和对流。  
- 黑/暗表面：吸收强、发射也强。  
- 保温设计本质是减少传导、对流、辐射中的一个或多个。
